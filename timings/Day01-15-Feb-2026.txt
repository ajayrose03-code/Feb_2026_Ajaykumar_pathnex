 ## Ansible Task - Install Nginx on Pathnex server

 Rewrite this YAML manually:


 ---
   -name: Install Nginx on Pathnex server
 	hosts: all
	become: yes

	tasks:
	   - name: Install nginx
	   yum:
	      name: nginx
	      state: present

    ## Jenkins Pipeline - checkout Git
    You will laern how to **checkout  a Git repository** and lists file.

    Pipeline {
         agent any
	 stages {
            stage ('Checkout') {
            steps {
            git branch: 'main', url: 'https://github.com/Pathnex/sample-java-app.git'
	   }
	 }
	 stage  ('List Files') {
	      steps{
	      sh 'ls -la'         
	 
	     }
	 }

      }
    }
